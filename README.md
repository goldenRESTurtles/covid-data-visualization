# covid-data-visualization
Investigate how vaccine coverage varies geographically and whether there is a relationship between the vaccination rate in a given area and a decrease in COVID-19 cases. It is hypothesized that a significant reduction in COVID-19 cases occurs once at least 50% of the population has been vaccinated at least once.
